ABOUT ME

I began my career as Software Engineer at General Assembly after graduating from San Francisco State University. I have always had a creative side, using foundational and technical artistic techniques. My experience successfully using software development tools such as Google Firebase led me to continue furthering my education in Software Engineering by enrolling in an immersive course to drive toward my goal of combining visual arts and technology. Recently, I have developed projects individually and collaboratively that focused on Full-Stack and Frontend Web Frameworks, and User-Interface.

Connect Four

Reconstructed a classic two-player turning it into a browser-based game using JavaScript algorithms and data structures with front-end libraries imported for design purposes.

 

Cafe DeLaSol

A user-centric Point of Sale System, it was necessary to implement Mongo database management and OAuth authentication for configuration and initialization. I implemented the Express framework and successfully created a web application using the Restful method.

 

Developers Meet

In cooperation with team members, I worked on the development of a full-stack Python web application with user-friendly features, with Amazon S3 authentication and the Django Web Framework. Deploying the completed application through Heroku.

 

NFT3: Social

Using NFTPorts' API, I developed a functional full-stack MERN application using the React web framework and synched it with MongoDB to store the user information and their preferred NFTs.
